话说我觉得精卫封闭式治疗对有自杀企图的患者完全是反作用

一开始我的抑郁也还好，但是经历了如此恐怖以至于可以说是受刑的治疗才毅然决然自杀

被剥夺自由甚至作为人的尊严，很大程度会压垮患者……

可以说，住院的几天是我人生最糟糕的几天，被绑在床上，切断通讯，没有事做，太恐怖了喵……

我自杀的时候感觉没有任何牵挂，已经被束缚到麻木了……昏迷后逐渐失去对身体的控制权，世界变得黑暗，甚至有一丝放松感

在ICU里看到了人们进进出出，可能有些人永远离开了，各种仪器的声音很吵很吵……所以说，自杀未遂的后果也会是很严重的喵，千万不要自杀喵！

现在我出院了超开心的哦！不是来自于治疗，而是来自于重回的自由

虽然我也没资格这么说，但希望有自杀企图或严重抑郁的人，多做做你们喜欢的事，不要放弃生命喵！
